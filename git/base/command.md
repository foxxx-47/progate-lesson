## Git の初期化
`$ git init`

## 共有するファイルの選択
`$ git add [ファイル名]`

## 選択したファイルを記録する
`$ git commit -m "メッセージ"`

## リモートリポジトリを登録する
`$ git remote add [リモートリポジトリ名] [URL]`

## リモートリポジトリにファイルをアップする
`$ git push origin master`

## リモートリポジトリのファイルをダウンロードする
`$ git pull origin master`

## 変更したファイルを把握する
`$ git status`

## 変更内容を把握しよう
`$ git diff`

## コミット履歴を確認する
`$ git log`
`$ git log -p` // 変更内容も一緒に確認する
